# TREKBASE HOLDING PAGE

## Install

  1 - Clone the repo
  2 - Create local DB with mongodb named trekbase
  3 - npm install
  4 - Ask for .env file

## Run

  - Dev (two terminals need to be run at the same time):
    1 - npm run build:dev:watch
    2 - npm run dev

  - Prod:
    1 - npm run build:prod
